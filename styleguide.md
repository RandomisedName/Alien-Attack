## Строки
```lua
-- Плохо
string = "hi"

-- Хорошо

```

## Переменные
```lua
-- Плохо

-- Хорошо

```

## Условные операторы
* Иначе с отрицаниями
```lua
-- Плохо
if not thing then
  -- ...код...
else
  -- ...код...
end

-- Хорошо
if thing then
  -- ...код...
else
  -- ...код...
end
```
* Правдивость
```lua
-- Плохо
if thing == true then
  -- ...код...
end

-- Хорошо
if thing then
  -- ...код...
end
```

## Пробелы
* Операторы
```lua
-- Плохо
name='John'

-- Хорошо
name = 'John'
```
* Запятые
```lua
-- Плохо
numbers = {1,2,3}
numbers = {1 , 2 , 3}
numbers = {1 ,2 ,3}

-- Хорошо
numbers = {1, 2, 3}
```
* В конце строк удалять пробелы
* В конце файла перенос строки
